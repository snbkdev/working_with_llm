# Змейка 🐍

Классическая игра «Змейка» на Python с использованием `pygame`.

## Установка и запуск

Используется общий `.venv` в корне репозитория:

```bash
source ../.venv/bin/activate    # активировать окружение
pip install -r ../requirements.txt   # если pygame ещё не установлен
python snake.py
```

## Управление

| Действие                | Клавиша / Кнопка                 |
|-------------------------|----------------------------------|
| Движение                | Стрелки / `W A S D`              |
| Пауза                   | `Пробел` или кнопка **Пауза**    |
| Рестарт после проигрыша | `R` или `Enter`                  |
| В меню после проигрыша  | `M`                              |
| Выбор сложности (в меню)| `1` / `2` / `3` или кнопки       |
| Выход                   | `Esc` или кнопка **Выход**       |

При запуске показывается стартовый экран с пояснением по управлению и
кнопкой **Начать** (или нажмите `Пробел` / `Enter`).

В верхней панели-меню отображаются текущий счёт, рекорд и кнопки
**Пауза** / **Выход** (кликабельные мышью). До старта в панели доступна
только кнопка **Выход**.

## Правила

- Управляйте змейкой и собирайте яблоки. Змейка растёт и **постепенно
  ускоряется** (множитель скорости показан в верхней панели).
- Игра заканчивается при столкновении с собственным телом (и со стеной —
  на сложном уровне).
- Звуковые эффекты: «бип» при поедании, трель за бонус и сигнал при
  проигрыше.
- **Фоновая музыка в стиле чиптюн**: 8-тактовая зацикленная мелодия
  (квадратная волна-соло как на NES, треугольный бас и ударные)
  синтезируется прямо при запуске — без аудиофайлов. На паузе музыка
  замолкает вместе с игрой.
- **Кнопки вкл/выкл** звука (динамик) и музыки (нота) — в верхней панели,
  работают и в меню, и во время игры. Выбор сохраняется между запусками
  в `settings.json`. Значения по умолчанию — `SOUND_ENABLED`,
  `MUSIC_ENABLED` и `MUSIC_VOLUME` в `config.py`.

## Уровни сложности

Выбираются на стартовом экране (кнопками или клавишами `1`/`2`/`3`):

| Уровень   | Темп       | Стены                          |
|-----------|------------|--------------------------------|
| Лёгкий    | медленный  | проходимы (выход с другой стороны) |
| Средний   | классика   | проходимы                      |
| Сложный   | быстрый    | **смертельны** (красная рамка) |

Режим записывается в таблицу рекордов отдельной колонкой.

## Комбо

Съедайте яблоки быстро — каждое яблоко, съеденное в течение 4 секунд
после предыдущего, повышает множитель: +1, +2, +3… (до ×9). Текущее
комбо и полоска-таймер показываются в правом верхнем углу поля.
Промедлили — множитель сгорает.

## Бонусы

**Каждые 5 яблок** на поле появляется случайный бонус. Он живёт 6 секунд
(кольцо-таймер вокруг иконки, в конце мигает):

| Бонус       | Эффект                                              |
|-------------|-----------------------------------------------------|
| ⭐ Звезда   | +5 очков, рост сразу на 2 клетки                    |
| 🐢 Черепаха | +2 очка, замедление на 10 секунд (передышка)        |
| ✂️ Ножницы  | +2 очка, отрезает половину хвоста (появляются только у длинной змейки) |
| 👻 Призрак  | +2 очка, 6 секунд можно проходить сквозь своё тело (змейка полупрозрачная) |

Активные эффекты показываются чипами с таймером в левом верхнем углу поля.

## Мышка и гнилое яблоко

После съеденного яблока на поле иногда появляются:

- **🐭 Мышка** (+4 очка) — живёт 12 секунд и **убегает от змейки**,
  перебегая на соседние клетки. Догони, если сможешь.
- **🍏 Гнилое яблоко** — лежит 8 секунд; съешь — **−3 очка, −2 клетки
  хвоста и сгорает комбо**. Над ним кружит муха — объезжай.

Легенда всей добычи с иконками показана на стартовом экране.

## Графика

- Плавное движение: змейка скользит между клетками (отрисовка 60 FPS,
  логика — отдельными шагами с интерполяцией).
- Змейка — округлое тело с градиентом от головы к хвосту и глазами,
  которые смотрят по направлению движения.
- Яблоко с бликом, листиком и мягким свечением, «дышит» пульсацией.
- Частицы-искры при поедании, всплывающие «+1» / «+5», вспышка и тряска
  экрана при проигрыше.

## Таблица рекордов

После проигрыша показывается таблица рекордов: **дата/время** и **счёт**.
Результаты сохраняются между запусками в файле `scores.json` (рядом с игрой,
хранится топ-20). Файл создаётся автоматически и исключён из git.

## Структура проекта

| Файл         | Назначение                                  |
|--------------|---------------------------------------------|
| `config.py`  | Настройки игры и цветовая палитра            |
| `scores.py`  | Таблица рекордов (чтение/запись `scores.json`) |
| `sound.py`   | Синтез звуковых эффектов (без аудиофайлов)   |
| `music.py`   | Чиптюн-музыка (синтез и зацикленное воспроизведение) |
| `settings.py`| Настройки звука/музыки (чтение/запись `settings.json`) |
| `button.py`  | UI-кнопка                                   |
| `effects.py` | Частицы и всплывающий текст                  |
| `game.py`    | Логика и отрисовка игры (класс `SnakeGame`)  |
| `snake.py`   | Точка входа (запуск)                         |

## Настройка

Параметры меняются в `config.py`:
- `COLS`, `ROWS` — размер поля, `CELL` — размер клетки в пикселях
- `DIFFICULTIES` — темп, ускорение и стены для каждого уровня сложности
- `BONUS_*`, `SLOW_*`, `GHOST_MS` — поведение бонусов
- `ROTTEN_*`, `MOUSE_*` — гнилое яблоко и мышка
- `COMBO_WINDOW_MS`, `COMBO_MAX` — комбо
- цвета (`*_COLOR` и др.) — оформление
