# Змейка 🐍

Классическая игра «Змейка» на Python с использованием `pygame`.

## Установка и запуск

Используется общий `.venv` в корне репозитория:

```bash
source ../.venv/bin/activate    # активировать окружение
pip install -r ../requirements.txt   # если pygame ещё не установлен
python snake.py
```

## Управление

| Действие                | Клавиша / Кнопка                 |
|-------------------------|----------------------------------|
| Движение                | Стрелки / `W A S D`              |
| Пауза                   | `Пробел` или кнопка **Пауза**    |
| Рестарт после проигрыша | `R`                              |
| Выход                   | `Esc` или кнопка **Выход**       |

При запуске показывается стартовый экран с пояснением по управлению и
кнопкой **Начать** (или нажмите `Пробел` / `Enter`).

В верхней панели-меню отображаются текущий счёт, рекорд и кнопки
**Пауза** / **Выход** (кликабельные мышью). До старта в панели доступна
только кнопка **Выход**.

## Правила

- Управляйте змейкой и собирайте еду (красные клетки) — за каждую +1 к счёту.
- Змейка растёт с каждой съеденной едой.
- **Стены проходимы**: выходя за край, змейка появляется с противоположной
  стороны экрана (как в классических мобильных «змейках»).
- Игра заканчивается **только** при столкновении с собственным телом.
- Звуковые эффекты: «бип» при поедании и сигнал при проигрыше. Звук можно
  отключить параметром `SOUND_ENABLED` в `config.py`.

## Таблица рекордов

После проигрыша показывается таблица рекордов: **дата/время** и **счёт**.
Результаты сохраняются между запусками в файле `scores.json` (рядом с игрой,
хранится топ-20). Файл создаётся автоматически и исключён из git.

## Структура проекта

| Файл         | Назначение                                  |
|--------------|---------------------------------------------|
| `config.py`  | Настройки игры и цветовая палитра            |
| `scores.py`  | Таблица рекордов (чтение/запись `scores.json`) |
| `sound.py`   | Синтез звуковых эффектов (без аудиофайлов)   |
| `button.py`  | UI-кнопка                                   |
| `game.py`    | Логика и отрисовка игры (класс `SnakeGame`)  |
| `snake.py`   | Точка входа (запуск)                         |

## Скорость и разгон

Змейка ускоряется по мере роста: стартует медленно и с каждой съеденной
едой становится чуть быстрее — до потолка. Текущая скорость показана в
верхней панели. Параметры в `config.py`:
- `SPEED_START` — начальная скорость (шагов/сек)
- `SPEED_STEP` — прирост за каждую еду
- `SPEED_MAX` — максимальная скорость

## Настройка

Прочие параметры в `config.py`:
- `COLS`, `ROWS` — размер поля
- `CELL` — размер клетки в пикселях
- цвета (`*_COLOR`) — оформление
