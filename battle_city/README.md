# Battle City 🛡️

Воссоздание классической игры **Battle City** (Tank 1990) с NES на Python +
`pygame`. Управляем танком, защищаем базу-орёл и разрушаем стены.

> Проект разрабатывается итерациями. Текущий статус и дорожная карта —
> в [`PLAN.md`](PLAN.md). Сейчас готов **Этап 1**: танк игрока, стрельба и
> разрушаемые стены (враги и условия победы — в следующих этапах).

## Установка и запуск

Используется общий `.venv` в корне репозитория:

```bash
source ../.venv/bin/activate          # активировать окружение
pip install -r ../requirements.txt    # если pygame ещё не установлен
python main.py
```

## Управление

| Действие   | Клавиша            |
|------------|--------------------|
| Движение   | Стрелки / `W A S D`|
| Огонь      | `Пробел`           |
| Рестарт    | `R`                |
| Выход      | `Esc`              |

## Правила (классические)

- Управляйте танком и защищайте **базу-орёл** в нижней части поля.
- **Кирпичные** стены разрушаются выстрелами, **стальные** — нет.
- Цель — уничтожить всех вражеских танков, не дав разрушить базу.

## Структура проекта

| Файл         | Назначение                                    |
|--------------|-----------------------------------------------|
| `config.py`  | Настройки, размеры, скорости, цвета            |
| `level.py`   | Карта поля, стены (кирпич/сталь), база-орёл    |
| `bullet.py`  | Пуля                                          |
| `tank.py`    | Танк (игрок/враг): движение, поворот, стрельба |
| `game.py`    | Игровой цикл, ввод, столкновения              |
| `main.py`    | Точка входа                                   |
| `PLAN.md`    | План разработки и статус этапов               |

## Настройка

Параметры меняются в `config.py`: размер поля (`COLS`, `ROWS`, `TILE`),
скорости (`PLAYER_SPEED`, `BULLET_SPEED`), стрельба (`PLAYER_SHOOT_COOLDOWN`,
`PLAYER_MAX_BULLETS`), параметры врагов и цвета (`*_COLOR`).
